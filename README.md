# DSAC Data Scientist Study Repository
DSAC 과정 참여 및 데이터 분석 실무 감각 향상을 목표로 정리한 개인 학습 레포지토리입니다.
- 본 저장소는 학교 프로그램을 통해 진행된 실습과 이론 내용을 정리


## 📘 목적
- DSAC 자격 취득 대비
- 데이터 분석 및 머신러닝 실습 기록
- 과정별 정리 및 복습용 자료 관리

---

### M1  
기초 이론 및 환경 세팅  
- 파이썬 기본 문법  
- 데이터 분석 기초  
- 라이브러리 실습

머신러닝 및 통계 분석 실습

- 3️⃣ 클러스터링  
- 4️⃣ 회귀분석 (선형회귀)  
- 5️⃣ 분류분석 (선형분류, 로지스틱회귀)
